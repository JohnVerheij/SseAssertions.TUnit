# Code of Conduct

This project adopts the [Contributor Covenant, version 3.0](https://www.contributor-covenant.org/version/3/0/code_of_conduct/) as its Code of Conduct. All contributors, maintainers, and participants in project spaces are expected to follow it.

## Reporting

To report a Code of Conduct concern, contact the project maintainer privately at [oss@verheij.io](mailto:oss@verheij.io). Reports are handled confidentially.

For security vulnerabilities, use the repository's [private security advisory channel](https://github.com/JohnVerheij/SseAssertions.TUnit/security/advisories/new) instead; see [SECURITY.md](./SECURITY.md) for the full vulnerability-disclosure policy.

## Enforcement

The maintainer is responsible for clarifying the standards described in the Contributor Covenant and is expected to take fair corrective action in response to any reported behavior that they deem inappropriate, threatening, offensive, or harmful. The maintainer has the right to remove, edit, or reject contributions that do not align with the Code of Conduct.
